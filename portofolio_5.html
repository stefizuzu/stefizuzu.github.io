3D Game Development in Unity

Project information

  • Category: Game Development
  • Software: Unity
  • Project GitHub: https://github.com/stefizuzu/Unity
  • Video URL: Link to YouTube

A 3-Dimensional shooting game, Capture the flag concept, with a client-server architecture. Game and server development in Unity with C#. The game was created in a team of 5 people. Unity has its own library to make it a multiplayer game but our team created its own hosting service on the server.
