Microsoft tablets management

Project for PwC

Project information

  • Category: Full-Stack Application
  • Tech stack: Java & JSP
  • Project: Link to YouTube

Developed a full-stack web application for PwC with two graphical user interfaces: one for the admin and one for the users. The admin interface was used as a control panel and the user interface was used to display the status of the tablets; if they were connected or not to the chargers. This application was intended for the workers at PwC.
The application was developed using Java, MySql on the back-end, and on the front-end JSP, CSS, HTML and JavaScript was used. Docker was also used to run the application.
