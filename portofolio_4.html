Email Distributed System

Project information

  • Category: Distributed System
  • Tech stack: #C & ASP.NET

A distributed email system in ASP.NET, using C# and the frameworks MVC and Entity Framework. Division into data layers (including Entity Framework), logic layers and presentation layers (eg using so-called view models). A login system has also been created. This web application was intended for a project at the Royal Institure of Technology.
